# EPI Manager 

### ⚠️ Contextualização do Problema
No SENAI SC Joinville Sul, funcionarios, alunos e visitantes utilizam EPIs para executar atividades e acessar areas especificas com seguranca.

Hoje, o controle de distribuicao desses equipamentos ainda e manual, feito com planilhas e relatorios em papel. Isso gera problemas recorrentes:

- Registros sujeitos a perdas, danos e erros.
- Dificuldade para consultar informacoes com rapidez.
- Falta de rastreabilidade sobre quem recebeu cada EPI.
- Estoque sem controle em tempo real.
- Reposicao sem alertas e pouca base para decisoes.

> A ausência de um sistema automatizado de controle de EPIs causa desorganizacao, falhas de estoque, baixa rastreabilidade e risco a protecao adequada dos usuarios.

### 🚧 Solucao Proposta
O EPI Manager centraliza e digitaliza o processo, substituindo registros manuais por um controle unico e confiavel.

Com ele, a instituicao passa a ter:

- Cadastro de usuarios e vinculo com EPIs.
- Registro de entregas e devolucoes.
- Controle de estoque mais preciso.
- Rastreabilidade e mais seguranca no uso.
- Dados para relatorios e apoio a gestao.

Assim, o projeto reduz falhas operacionais e torna a gestao de EPIs mais segura, eficiente e organizada.

### ⚒️ Instalacao e Configuracoes Necessarias

#### Pre-requisitos
- Node.js (versao LTS recomendada)
- npm (instalado com o Node.js)
- Git (opcional, para clonar o repositorio)

#### Instalacao
```bash
npm install
```

#### Executar em desenvolvimento
```bash
npm run dev
```

Depois, acesse no navegador o endereco informado no terminal (normalmente http://localhost:5173).
